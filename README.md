# react-native-bob-design

React Native Bob Design System

## Installation

Add in package.json

```json
"react-native-bob-design": "git+ssh://git@gitlab.com:multinode/md/react-native-bob-design.git#0.1.5"
```

## Usage
```ts
import {InformationBox, CustomRadio, InformationBox} from 'react-native-bob-design'
```
### Components
- InformationBox
- CustomCheckbox
- CustomRadio
- Input
- PasswordInput
- PhoneInput
- Textarea
